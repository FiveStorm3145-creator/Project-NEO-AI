# Neo Backend

This directory is reserved for the future Python FastAPI backend.

Backend implementation is intentionally not included yet. When Neo is ready for server-side features, this area can host API routes for assistant orchestration, memory, automation, and voice service integrations.

Planned principles:

- Keep API routes small and explicit.
- Do not commit secrets or local credentials.
- Keep AI and voice integrations behind clear service boundaries.
- Add tests before introducing backend behavior.
